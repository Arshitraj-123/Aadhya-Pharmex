# Aadhya Pharmex Distribution DMS

A comprehensive, responsive Pharmaceutical Distribution Management System (DMS) built with React, Vite, and Tailwind CSS. 

## Overview
Aadhya Pharmex DMS is designed to streamline pharmaceutical supply chains, offering complete control over:
- Live Inventory and Stock Allocation
- Real-time Delivery Tracking and Dispatch Control
- Retailer Network Management
- GST and Regulatory Compliance

## Features
- **High-End Animated Auth Gateway:** Beautifully styled split-screen login and registration pages using `framer-motion` for buttery smooth transitions, nested spring animations, and micro-interactions.
- **Responsive Dashboard Layouts:** Adaptive sidebar and widget grids constructed natively with CSS and modern utility classes.
- **Tailwind CSS Integration:** Configured with a deep, branded Navy Blue and Success Green palette matching the Aadhya Pharmex visual identity.
- **Iconography:** Powered by Lucide React.

## Tech Stack
- **Framework:** React 19 + Vite 6
- **Styling:** Tailwind CSS v3
- **Animations:** Framer Motion
- **Icons:** Lucide React

## Getting Started

### Prerequisites
- Node.js (v18+ recommended)
- npm or yarn

### Installation
1. Clone the repository
   ```bash
   git clone https://github.com/Arshitraj-123/Aadhya-Pharmex.git
   ```
2. Navigate to the project directory
   ```bash
   cd Aadhya-Pharmex
   ```
3. Install dependencies
   ```bash
   npm install
   ```
4. Start the development server
   ```bash
   npm run dev
   ```

### UI & Styling System
The application utilizes a custom color hierarchy defined in `tailwind.config.js`:
- `brand-dark`: The deep navy blue sidebar backdrop (`#27374D`)
- `status-success-solid`: The signature green branding color (`#10B981`)
- Standard Tailwind `slate` scales for typographic hierarchy and borders.
